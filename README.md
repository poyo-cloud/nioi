# 匂い研究スケジュール

外部の計測機器で確認した時刻を入力すると、研究手順の次の指示時刻を算出する iPhone 向けの軽量 Web アプリです。

## できること

- パターン A〜H の選択
- 対照実験として最初に 3分間の安静測定を追加
- 条件 1〜7 の順番管理
- 各条件で「ストレス課題開始時刻」を入れると、3分のストレス課題と、その後の呼吸測定時刻を表示
- 呼吸測定開始時刻を空欄にした場合は、ストレス課題開始 + 3分10秒 を自動で使用
- 呼吸測定開始が 10秒以上ずれた場合でも、手動で修正可能
- 呼吸測定後の 0秒 / 30秒 / 60秒 / 90秒 の指示時刻と 120秒の終了時刻を表示
- 条件ごとのメモ記録
- 区間時刻つき CSV の書き出し
- 保存済み履歴からの CSV 再書き出し
- 生データとの突き合わせや平均値計算は別ツールで行う前提
- 端末内のローカル履歴保存

## 使い方

1. `index.html` をブラウザで開く
   Safari なら共有メニューから「ホーム画面に追加」で iPhone から使いやすくできます
2. パターンを選ぶ
3. まず対照安静 3分の開始時刻を入力して `指示を作成`
4. 各条件では、ストレス課題開始時刻を入力して `指示を作成`
5. 呼吸測定開始が遅れた場合は、呼吸測定開始時刻を手入力で上書きする
6. 実施後に必要ならメモを書き、`この条件を記録して次へ`
7. すべて終わったら、別ツールへ渡すための CSV を書き出す
8. 過去セッションは履歴欄から再度 CSV を書き出せます

## 前提

- 対照実験は 3分間の安静測定として扱います
- 各条件は「ストレス課題 3分」+「インターバル 10秒以上」+「呼吸測定 120秒」で扱います
- 呼吸測定の 4 区間は、あとで別ツールで平均値抽出しやすい形で CSV に出力します
- アプリ自体ではタイマー計測を行いません
- 保存履歴はこのブラウザの `localStorage` に保存されます
