# 匂い研究スケジュール

外部の計測機器で確認した時刻を入力すると、研究手順の次の指示時刻を算出する iPhone 向けの軽量 Web アプリです。

## できること

- パターン A〜H の選択
- 条件 1〜7 の順番管理
- 呼吸開始時刻から 0秒 / 30秒 / 60秒 / 90秒 の指示時刻と 120秒の終了時刻を表示
- 条件ごとのメモ記録
- 7 条件完了後のメモ生成
- 端末内のローカル履歴保存

## 使い方

1. `index.html` をブラウザで開く
   Safari なら共有メニューから「ホーム画面に追加」で iPhone から使いやすくできます
2. パターンを選ぶ
3. 呼吸を開始するたびに、計測機器の時刻を分・秒で入力して `指示を作成`
4. 実施後に必要ならメモを書き、`この条件を記録して次へ`
5. 7 条件が終わると履歴へ自動保存されるので、必要なら完了メモをコピーする

## 前提

- 画像に合わせて、呼吸開始後の 4 指示を「0秒 / 30秒 / 60秒 / 90秒」で表示します
- あわせて 120秒時点の「終了」も表示します
- アプリ自体ではタイマー計測を行いません
- 保存履歴はこのブラウザの `localStorage` に保存されます
